# Developer Landing Page

This project is a landing page designed for a software developer, built using React and styled with Tailwind CSS. The theme of the landing page is primarily blue, providing a clean and modern look.

## Features

- **Responsive Design**: The landing page is fully responsive and looks great on all devices.
- **Components**: The project is structured with reusable components such as Navbar, Hero, About, Projects, Contact, and Footer.
- **Custom Hook**: A custom hook `useTheme` is included to manage theme states (light/dark mode).
- **TypeScript**: The project is built with TypeScript for type safety and better development experience.

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd developer-landing-react
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the application**:
   ```
   npm start
   ```

4. **Open your browser**:
   Navigate to `http://localhost:3000` to view the landing page.

## Customization

You can customize the theme and styles by modifying the `tailwind.config.js` file and the `src/styles/index.css` file.
